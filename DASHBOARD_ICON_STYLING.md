## Statistics Cards with Circular Icon Badges

### Implementation

Each statistics card has a **circular colored background** with a **filtered SVG icon** in the center.

```html
<div class="db-stat-card">
    <div class="db-stat-icon blue">
        <img src="~/images/icons/Icon-3.svg" alt="Voyage Reports" class="stat-icon-img" />
    </div>
    <div class="db-stat-body">
        <div class="db-stat-label">Voyage Reports</div>
        <div class="db-stat-num">6</div>
        <div class="db-stat-change green">+17% Since yesterday</div>
```

### Color Schemes

#### 1. Blue Theme (Voyage Reports)
- **Background**: `#dbeafe` (light blue)
- **Icon Color**: `#3b82f6` (blue) via CSS filter
- **Icon**: `Icon-3.svg` (Document icon)
- **Size**: 56×56px circle

```css
.db-stat-icon.blue {
    background: #dbeafe;
}

.db-stat-icon.blue .stat-icon-img {
    filter: brightness(0) saturate(100%) invert(32%) sepia(96%) saturate(1831%);
}
```

#### 2. Red Theme (Incidents)
- **Background**: `#fee2e2` (light red)
- **Icon Color**: `#ef4444` (red) via CSS filter
- **Icon**: `Icon-2.svg` (Warning icon)
- **Size**: 56×56px circle

```css
.db-stat-icon.red {
    background: #fee2e2;
}

.db-stat-icon.red .stat-icon-img {
    filter: brightness(0) saturate(100%) invert(28%) sepia(82%) saturate(2876%);
}
```

#### 3. Orange Theme (Unsubmitted)
- **Background**: `#fed7aa` (light orange)
- **Icon Color**: `#f59e0b` (orange) via CSS filter
- **Icon**: `Icon-17.svg` (Exclamation icon)
- **Size**: 56×56px circle

```css
.db-stat-icon.orange {
    background: #fed7aa;
}

.db-stat-icon.orange .stat-icon-img {
    filter: brightness(0) saturate(100%) invert(50%) sepia(98%) saturate(1773%);
}
```

## CSS Filter Color Reference

### How CSS Filters Work

SVG icons are black by default. We use CSS `filter` to change their color:

```css
/* Black ? White */
filter: brightness(0) invert(1);

/* Black ? Blue (#3b82f6) */
filter: brightness(0) saturate(100%) invert(32%) sepia(96%) saturate(1831%) hue-rotate(203deg) brightness(101%) contrast(97%);

/* Black ? Red (#ef4444) */
filter: brightness(0) saturate(100%) invert(28%) sepia(82%) saturate(2876%) hue-rotate(346deg) brightness(95%) contrast(98%);

/* Black ? Orange (#f59e0b) */
filter: brightness(0) saturate(100%) invert(50%) sepia(98%) saturate(1773%) hue-rotate(359deg) brightness(101%) contrast(101%);
```

## Responsive Behavior

### Desktop (>1024px)
- 3-column statistics grid
- 2-column charts grid
- 2-column bottom section (1.6fr + 1fr)

### Tablet & Mobile (<1024px)
- Single column stacking
- Icons remain same size for clarity
- Cards take full width

```css
@media (max-width: 1024px) {
    .db-stats, .db-charts, .db-bottom {
        grid-template-columns: 1fr;
    }
}
```

## Accessibility

All icons follow accessibility best practices:

1. **Alt text**: Empty `alt=""` for decorative icons with adjacent text
2. **Color contrast**: WCAG AA compliant (4.5:1 minimum)
3. **Keyboard navigation**: All interactive elements are focusable
4. **Screen readers**: Text labels provide context

```html
<!-- Decorative icon with adjacent text -->
<img src="~/images/icons/Icon-3.svg" alt="" class="stat-icon-img" />
<div class="db-stat-label">Voyage Reports</div>

<!-- Standalone icon needs description -->
<img src="~/images/icons/Icon-15.svg" alt="Flight icon" />
```
